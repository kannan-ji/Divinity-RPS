# Divinity Original Sin RPS Predictor

A web application designed to help players dominate Rock-Paper-Scissors mini-games in *Divinity: Original Sin*.

## Description
Master the art of persuasion and debate in *Divinity: Original Sin*! This RPS Predictor analyzes NPC behavior and suggests the most likely winning moves based on historical patterns, giving you the upper hand in crucial persuasion checks.

## Key Features
- **RPS Pattern Analysis:** Track previous moves to predict upcoming choices.
- **Minimalist UI:** Designed to be lightweight and unobtrusive while playing.
- **Easy Sharing:** Built-in sharing functionality to challenge friends or share your winning streaks.
